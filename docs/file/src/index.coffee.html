Method Details

~ (void) createdBy(schema, options = {})

The createdBy method adds a field to a schema to store the userId of the user who created this object. Please note that there are two versions supported. The original used the concept of an actor object, but it turned out to be overkill. The new one is simpler, it just addes a createdByUserId field of type ObjectId

To run with the old version: Do nothing. To run with the new version: set options.v to 2 To not keep both fields: set options.keepV1 to false
